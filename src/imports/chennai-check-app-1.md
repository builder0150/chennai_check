Create a playful mobile interactive web experience with 4 screens.

Tone: Cute, dramatic, playful, slightly emotionally manipulative but funny. Gen Z energy.

Design Style:
- Minimal UI
- Bold typography
- Smooth animated transitions
- Dynamic color shifts based on emotional state
- Top happiness meter bar labeled “Navya’s Happiness Index”
- Confetti and meme-style micro animations

Flow:

Screen 1 – Landing:
Title: “Operation: Chennai Check 💌”
Subtitle: “Your answer may directly affect my happiness levels.”
Start button: “Start the interrogation 😌”

Screen 2 – Question 1:
Question: “Will you be coming to Chennai this Saturday?”
Options:
YES 😌
NO 😐

If YES:
- Happiness meter moves to maximum
- Warm yellow/pink glow
- Confetti burst
- Show text: “Elite decision detected. My happiness just did a backflip.”
- Move to final screen

If NO:
- Background fades to black
- Happiness meter moves left to sad
- Show text: “Oh. That was… unexpected.”
- Then load next question

Screen 3 – Question 2:
Question: “Then when are you coming? 🤨”
Options:
1. “This weekend. Obviously. I was joking.”
2. “Next Monday. Slight delay, still loyal.”
3. “There is no other option. I’m coming this weekend.”

Option logic:
- Option 1 → Happiness full
- Option 2 → Happiness medium
- Option 3 → Happiness full with big celebration animation

Final Screen:
Big text: “COOL. SEE YOU SOON. 🥳”
Confetti animation
Replay button
